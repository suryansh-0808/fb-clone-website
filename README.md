# fb-clone-website

A simple Facebook clone website developed as part of a Git and GitHub learning tutorial. This project was created to understand the fundamentals of version control, repository management, and collaborative development using Git.

The website recreates the basic look and feel of Facebook's homepage using front-end web technologies. It serves as a practice project for improving HTML, CSS, and JavaScript skills while also gaining hands-on experience with Git commands and GitHub workflows.

## Features

- Facebook-inspired user interface
- Responsive and clean layout
- Built using HTML, CSS, and JavaScript
- Well-structured project files
- Beginner-friendly codebase

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## Learning Objectives

This project was created to practice and understand:

- Initializing a Git repository
- Tracking and committing changes
- Creating and managing branches
- Merging branches
- Resolving merge conflicts
- Pushing code to GitHub
- Cloning repositories
- Working with remote repositories

## Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/fb-clone-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd fb-clone-website
   ```

3. Open the `index.html` file in your browser or use a live server extension in Visual Studio Code.
